# Мемный Echo Bot

Простой Telegram-бот для теста Git pull.

## Команды:
- /start — главное меню
- /joke — случайная шутка
- Просто пиши сообщения

## Запуск:
1. `pip install -r requirements.txt`
2. Вставь токен в `bot.py`
3. `python bot.py`

Лёгкий код специально для быстрого тестирования изменений.
